## Login issues

**I cannot sign in**
- Confirm you are using the correct email address (case-sensitive)
- Check that your account has not been deactivated — ask your Admin
- If your password is wrong, ask an Admin to reset it (there is no self-service password reset)

**I see a blank page after signing in**
- Clear your browser cache and reload
- Try a different browser or private/incognito window
- Contact your admin if the issue persists

---

## Client and account issues

**I cannot find a client I just created**
- The client list loads on page render — refresh the page
- Check that you did not accidentally navigate away before saving the form

**I accidentally created a duplicate client**
- Update one record to the correct information
- Contact your Admin to delete the duplicate

**I cannot link a debt account to a creditor**
- Search by the creditor's full legal name, not an abbreviation
- If the creditor does not exist, create it first (Creditors → New Creditor), then return to the account

---

## Negotiation activity issues

**My activity was saved but the Last Contact Date did not update**
- Last Contact Date only updates for contact-type activities: Call, Voicemail, Email, Letter, Offer Sent, Offer Received
- Internal Note and Client Update types do not update Last Contact Date by design

**I set a Next Action Date but the task was not auto-created**
- Auto-task creation requires you to check the "Create follow-up task" checkbox on the activity form
- If you forgot, create the task manually from the Tasks section

---

## Offers

**I cannot accept a second offer on an account**
- Only one offer can be Accepted per account at a time
- Change the first accepted offer to a different status (Expired or Countered) before accepting the new one
- Or confirm with the system prompt whether to replace the prior accepted offer

---

## Documents

**My file upload failed**
- Maximum file size is 4 MB — compress large PDFs before uploading
- Only these types are accepted: PDF, JPG, PNG, WebP, DOC, DOCX, TXT
- Check your internet connection; uploads require a stable connection

**I cannot download a document**
- You must be signed in to download — confirm your session has not expired
- If the download shows an error, the file may have been deleted; contact your Admin

---

## Dashboard

**The Dashboard shows 0 follow-ups but I know tasks exist**
- Tasks only appear on the Dashboard if they have a due date set
- Check the Tasks page and add due dates to tasks that are missing them

**Overdue tasks count is very high**
- This usually means tasks are being created but not completed or rescheduled
- Set aside time to review and triage the task list weekly

---

## General tips

- **Reload the page** after any data operation if you do not see your changes reflected immediately
- **Use full legal names** for creditors — abbreviated names cause duplicate creditor records
- **Log activities immediately** — waiting until end-of-day causes missed details and incorrect contact dates
