## What is a negotiation activity?

A negotiation activity is any interaction or event related to a debt account — a phone call, an email, a letter, a settlement discussion, or an internal note. Every activity is time-stamped and tied to a specific debt account.

Logging activities consistently is critical for:
- Keeping an accurate timeline for compliance
- Knowing when to follow up with a creditor
- Giving team members visibility into what has already happened

## How to log an activity

1. Navigate to the **Debt Account** detail page (Clients → client → debt account)
2. Scroll to the **Negotiation Activity** section
3. Click **Log Activity** or use the inline form
4. Select the **Activity Type** (see below)
5. Write detailed **Notes** — describe exactly what happened or was communicated
6. Set the **Date/Time** — defaults to now, adjust if logging retroactively
7. (Optional) Set a **Next Action Date** — if follow-up is needed, set when
8. (Optional) Check **Create follow-up task** — this will auto-create a task linked to this account
9. Click **Save Activity**

## Activity types

| Type | When to use |
|------|-------------|
| **Call** | Outbound or inbound phone call with creditor |
| **Voicemail** | Left a voicemail for the creditor |
| **Email** | Sent or received an email |
| **Letter** | Mailed or received a physical letter |
| **Internal Note** | Team-only note not related to a specific contact |
| **Client Update** | Communication with the client themselves |
| **Settlement Discussion** | Any discussion specifically about a settlement offer |
| **Status Change** | Recording a change to the account status |
| **Offer Sent** | Sent a formal offer to the creditor |
| **Offer Received** | Received a formal offer from the creditor |

## Auto-updates after logging

- **Last Contact Date** on the debt account is automatically updated for contact-type activities (Call, Voicemail, Email, Letter, Offer Sent/Received)
- **Next Follow-Up Date** on the debt account is automatically updated if you set a Next Action Date

## Tips

- Log activities immediately after they happen — never let them pile up
- Use **Settlement Discussion** type for any conversation about numbers, even informal
- Set a Next Action Date on every contact activity — this keeps the dashboard accurate
- Use Internal Note for team coordination that should not appear as a creditor interaction

## Common mistakes

- Writing vague notes like "Called creditor" — always record what was said, next steps, and any creditor reference numbers
- Forgetting to set the Next Action Date after a call — this causes items to fall off the Dashboard
- Using Internal Note for creditor contacts — use the correct type so Last Contact Date updates
