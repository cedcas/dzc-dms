# DMS Brand Kit

Files included:
- `main-logo.svg` — primary full logo for headers and marketing-style placements
- `login-logo.svg` — centered logo lockup for the login screen
- `sidebar-icon.svg` — icon inside rounded square for app sidebar/app shell
- `favicon.svg` — simplified favicon-safe SVG
- `icon-plain.svg` — plain icon without background container

Primary colors:
- Navy: #1E3A8A
- Blue: #1D4ED8
- Cyan: #06B6D4
- Teal accent: #14B8A6

Suggested usage:
- Header / app shell: `main-logo.svg`
- Sidebar expanded: `sidebar-icon.svg` or `icon-plain.svg` + text
- Sidebar collapsed: `sidebar-icon.svg`
- Login screen: `login-logo.svg`
- Favicon: `favicon.svg`
